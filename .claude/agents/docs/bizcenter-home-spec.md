## 섹션별 기능 정의

### [상단] 회사 정보
- **노출**: 기업명, 기업로고, 더보기 버튼
- **링크**: 기업정보 페이지 이동
- **케이스**: 항상 노출 (디폴트)

---

### [상단] 회원 정보
- **노출**: 아이디, 기업인증 상태, 권한 상태
- **링크**: 기업인증 페이지 이동

| Case | 조건 | 상태 표시 |
|------|------|---------|
| Case 1 | 기업인증 미인증 | "기업인증 후 공고등록 가능!" 팝업 노출 → 클릭 시 기업인증 페이지 이동 |
| Case 2 | 기업인증 심사중 | 심사중 상태 노출 → 클릭 시 기업인증 페이지 이동 |
| Case 3 | 기업인증 성공 | 성공 상태 노출 → 클릭 시 기업인증 페이지 이동 |
| Case 4 | 기업인증 실패 | 실패 상태 노출 → 클릭 시 기업인증 페이지 이동 |
| Case 5 | 마스터인 경우 | 마스터인 상태 노출 |
| Case 6 | 일반 멤버인 경우 | 일반멤버 상태 노출 |
| Case 7 | 간편 멤버인 경우 | 간편멤버 상태 노출 |

---

### [상단] 권한 관리 (마스터 전용)
- **링크**: 초대 관리 페이지 이동

| Case | 조건 | 노출 내용 |
|------|------|---------|
| Case 1 | 마스터가 아닌 경우 | **영역 미노출** |
| Case 2 | 마스터 & 승인 전 초대한 사용자 있음 | 카운트 노출 → 클릭 시 초대 관리 페이지 |
| Case 3 | 마스터 & 승인 전 초대한 사용자 없음 | 숫자 0 노출 → 클릭 시 초대 관리 페이지 |
| Case 4 | 마스터 & WS에 멤버 있음 | 카운트 노출 → 클릭 시 초대 관리 페이지 |
| Case 5 | 마스터 & WS에 멤버 없음 | "멤버를 초대해 보세요!" 팝업 + 초대하기 버튼 |

- **상세 분류**: 요약 / 채용 리포트

|------|---------|------|

| 채용 리포트 | 채용결과보고서 명, 다운로드 버튼, 더보기 버튼 | 리포트 페이지 이동 |

### [중단] 비즈머니 — 유료서비스 이용 현황
- **링크**: 비즈머니 내역 이동

| Case | 조건 | 노출 내용 |
|------|------|---------|
| Case 1 | 비즈머니 있음 | (유료 상태의) 보유 비즈머니 노출 |
| Case 2 | 비즈머니 없음 | 문구 노출 + 충전하기 버튼 |
| Case 3 | 무료 포인트 있음 | (유료 상태의) 보유한 무료 포인트 노출 |
| Case 4 | 무료 포인트 없음 | 하이픈(`-`) 처리 |
| Case 5 | 소멸 예정 포인트 있음 | 1개월 이내 소멸 예정 포인트 노출 |
| Case 6 | 소멸 예정 포인트 없음 | 하이픈(`-`) 처리 |

### [중단] 공고 현황

| 분류 | 케이스 | 노출 내용 | 링크 |
|------|--------|---------|------|
| 채용 진행중 공고 | Case 1: 진행중 공고 있음 | 공고 카운트 & 최신순 최대 N개 리스트 (지원자수, 담당자명, 결제 상품, 상품 추천) | 게재중 리스트 |
| 채용 진행중 공고 | Case 2: 진행중 공고 없음 (등록 이력 있음) | 공고등록 유도 문구 + 공고등록 버튼 | — |
| 채용 진행중 공고 | Case 3: 공고 등록 0건 | 공고등록 유도 문구 + 공고등록 버튼 | — |

### [중단] 채용 진행중 지원자 (ATS)
- **탭 구성**: 미열람 / 면접일정조정 / 면접종료 / 미심사 (4개 탭)
- **공통 필터**: 공고 마감 90일 경과 OR 영상기간만료 OR 채용취적합 지원자 미노출
- **카드 노출 정보**: 사진, 신입/경력 여부, 이름, 성별, 나이, 거주지, 가장 최근 경력, 지원한 공고 정보(제목)
- **링크**: 이력서 뷰 이동

| Case | 조건 | 동작 |
|------|------|------|
| Case 1 | 4개 탭 모두 지원자 없음 | **영역 미노출** |
| Case 2 | 미열람 탭 1명 이상 있음 | 1개 탭 + 지원자 최대 N명 노출 |
| Case 3 | 미열람 없음 | '심사중' 상태 & 아래서 미열람 지원자 리스트 / 지원일 오래된 순 |
| Case 4 | 면접일정조정 지원자 있음 | '면접일정조정' 상태 리스트 / 면접 일정 변경 요청 오래된 순 |
| Case 5 | 면접종료 지원자 있음 | '면접수락' 상태 & 현재일시+면접일시 리스트 / 면접종료 오래된 순 |
| Case 6 | 미심사 지원자 없음 | 심사중·면접제안·면접일정조정·면접수락 상태 지원자 / 지원일시 오래된 순 |

---

### [중단] 면접캘린더 (ATS)
- **링크**: 면접캘린더 페이지 이동

| Case | 조건 | 노출 내용 |
|------|------|---------|
| Case 1 | 면접일정 설정된 지원자 있음 | 당일 날짜(MM.DD) + 해당 일 면접 설정 지원자수 + 캘린더 바로가기 버튼 |
| Case 2 | 면접일정 설정된 지원자 없음 | 당일 날짜(MM.DD) + 0건 + 캘린더 바로가기 버튼 |

---

### [중단] 지원자 면탁 (ATS)
- **노출**: 지원자 별 인사담당자 메시지, 면접 제안, 결과 안내
- **링크**: 이력서 뷰 이동

---

### [하단] 공지사항
- **1월 런칭 버전**: 노선 공지사항 고정 노출 (제목, 등록일시)
- **케이스**: 노선에 연결된 공지사항 없음 시 해당 공지 고정 표시
- **링크**: 노선 공지사항 페이지 이동

---

### [하단] FAQ
- **1월 런칭 버전**: 노선 FAQ N개 고정 노출
- **케이스**: 노선에 연결된 FAQ 없음 시 고정 표시
- **링크**: 노선 FAQ 페이지 이동

---

### [하단] 채용컨설턴트/고객센터

| Case | 조건 | 노출 내용 |
|------|------|---------|
| Case 1 | 배정된 RM 담당자 있음 | RM 이름, 전화번호, 이메일, 상담시간 |
| Case 2 | 배정된 RM 담당자 없음 | 고객센터 전화번호, 이메일, 상담시간 |

## 기획 에이전트 사용 가이드

- **pm-lead**: 이 문서를 PRD 작성 시 기능 요구사항 베이스로 사용
- **spec-writer**: 각 섹션의 케이스 테이블을 FR(기능 요구사항)으로 변환
- **mock-generator**: Case별 시나리오 Mock 데이터 생성 기준으로 활용
- **qa-checker**: 케이스 누락 여부 검증 기준으로 활용
