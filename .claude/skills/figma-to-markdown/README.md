# 📝 Figma to Markdown Skill

Figma 기획서를 구조화된 UI 로직 마크다운 문서로 자동 변환하는 Claude Code 스킬입니다.

## 🚀 사용 방법

### 기본 사용

```bash
/figma-to-markdown <figma-url>
```

### 예시

```bash
# 디자인 파일
/figma-to-markdown https://figma.com/design/uy4pUNV15IRjwQ7SL3FaO8/인재검색-상세?node-id=781-26433

# FigJam 보드
/figma-to-markdown https://figma.com/board/abc123/기획서?node-id=1-2
```

## 📋 기능

이 스킬은 Figma 기획서에서 다음 정보를 자동으로 추출합니다:

1. **요구사항** - 체크리스트, 번호 리스트, 기능 목록
2. **상태 플로우** - 상태 정의 및 전환 규칙 (Mermaid 다이어그램 포함)
3. **UI 구조** - 상태별 표시 요소, 버튼, 액션
4. **정보 노출 정책** - 필드별 접근 규칙 (수락 전/후 구분)
5. **비즈니스 로직** - 검증 규칙, 권한 체크

## 📄 출력 결과

생성되는 파일 위치:

```
docs/ui-logic/{feature-name}.md
```

예시:

- `docs/ui-logic/talent-detail-resume-view.md`
- `docs/ui-logic/applicant-management.md`
- `docs/ui-logic/job-posting-form.md`

## 🎯 문서 작성 원칙

이 스킬은 다음 원칙을 따라 문서를 생성합니다:

### ✅ 포함되는 것

- 기능적/행동적 명세
- 상태 전환 규칙
- 접근 제어 정책
- 검증 규칙

### ❌ 제외되는 것

- 디자인 요소 (색상, 폰트, 스타일)
- 추측된 내용 (→ TODO로 표시)
- 중복된 정보

### 📝 품질 기준

1. **정확성**: Figma에 명시된 내용만 포함
2. **간결성**: 중복 없이 한 번만 설명
3. **구조화**: 카테고리 → 필드 → 조건별 규칙
4. **명시성**: 불확실한 내용은 TODO로 표시

## 🔄 작업 흐름

```
Figma URL 입력
  ↓
URL 파싱 (fileKey, nodeId 추출)
  ↓
Figma MCP로 데이터 가져오기
  ↓
내용 분석 및 추출
  ├─ 요구사항
  ├─ 상태 플로우
  ├─ UI 구조
  └─ 정보 노출 정책
  ↓
마크다운 생성
  ↓
파일 저장 (docs/ui-logic/*.md)
  ↓
완료 (검토 및 수정 안내)
```

## ⚠️ 생성 후 확인사항

문서 생성 후 다음 항목을 확인하세요:

1. **TODO 항목 처리** - 불확실한 내용 검증
2. **디자인 요소 제거 확인** - 색상, 스타일 명세 없는지 확인
3. **중복 내용 확인** - 같은 정보가 여러 곳에 없는지 확인
4. **구조 검토** - 카테고리 우선 → 필드별 → 조건별 구조 확인
5. **Figma 링크 확인** - 원본 기획서 접근 가능한지 확인

## 💡 사용 팁

### Tip 1: 정확한 node-id 사용

Figma에서 특정 프레임/섹션을 선택하고 URL을 복사하면 정확한 node-id를 얻을 수 있습니다.

### Tip 2: 반복 실행

문서가 마음에 들지 않으면 다시 실행하세요. 기존 파일은 덮어씌워집니다.

### Tip 3: 점진적 개선

1차 생성 후 TODO 항목을 직접 채우고, 필요한 섹션을 추가하세요.

### Tip 4: 템플릿 활용

생성된 문서를 팀 내 템플릿으로 활용할 수 있습니다.

## 🐛 문제 해결

### Q: "node-id 파라미터가 없습니다" 에러

Figma URL에 `?node-id=123-456` 형식의 파라미터가 포함되어 있는지 확인하세요.

### Q: 생성된 문서가 비어있어요

Figma 기획서의 구조가 스킬이 인식하는 패턴과 다를 수 있습니다. TODO 항목을 참고해서 수동으로 채워주세요.

### Q: 디자인 요소가 포함되어 있어요

피드백을 주시면 스킬을 개선하겠습니다. 일단 수동으로 제거해주세요.

- 파싱 오류가 있나요?
