## 사용 방법

1. 각 항목을 하나씩 확인하며 체크
2. 불일치 항목 발견 시 상세 내용 기록
3. 심각도 분류 (🔴/🟡/🟢)
4. 수정 권장사항 작성
5. 수정 후 재검수
