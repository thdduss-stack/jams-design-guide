## 1. 서비스 개요

잡코리아의 유료 채용광고 상품으로, **PC 및 모바일 웹/앱에 동시 노출**되는 서비스입니다.

### 1.1 상품 카테고리


| 카테고리      | 상품 수 | 특징                   |
| --------- | ---- | -------------------- |
| 메인 채용관    | 4종   | PC + 모바일 메인 최상단 노출   |
| MAX 결합 상품 | 7종   | 직무·산업 + 근무지 지역 동시 노출 |
| 직무·산업 전용  | 7종   | 직무·산업 리스트 노출         |
| 지역 전용     | 7종   | 근무지 지역 리스트 노출        |
| 점프업       | 3종   | 무료공고 리스트 상단 점프업      |


### 1.2 공통 조건

- **이용 기간**: 2일 ~ 30일 (최소 2일 이상 신청)
- **가격 기준**: 일일 단가 (VAT 포함)
- **노출 범위**: PC + 모바일 웹/앱 동시 노출

## 6. 점프업 상품

잡코리아 **채용정보 리스트 + 모바일 웹/앱 리스트**에서 PC/모바일 동시 점프업하는 상품입니다.
무료공고 리스트 상단으로 올라가 노출 효과를 극대화합니다.

### 6.1 가격표


| 옵션      | 일일 가격 (VAT 포함) | 설명     |
| ------- | -------------- | ------ |
| 3회 점프업  | **30,800원/일**  | 기본 점프업 |
| 6회 점프업  | **38,500원/일**  | 효율 점프업 |
| 12회 점프업 | **49,500원/일**  | 집중 점프업 |


### 6.2 기능 상세

- **점프업 동작**: 무료공고 리스트에서 PC/모바일 동시에 상단으로 점프업
- **채용속보 노출**: 채용속보에 랜덤으로 노출
- **이용 기간**: 2~30일

### 6.3 점프업 안내

무료공고 리스트에서 PC/모바일 동시 점프업되며, 채용속보에 랜덤으로 노출됩니다.
하루 동안 설정한 횟수만큼 자동으로 점프업이 실행됩니다.

## 8. 신청 절차 및 조건

### 8.1 신청 조건

- **기업 로그인 필수**: 잡코리아 기업 회원으로 로그인해야 합니다
- **채용공고 등록 필수**: 진행 중인 채용공고가 있어야 유료서비스 신청이 가능합니다
- **최소 신청 기간**: 2일 이상

### 8.2 신청 절차

1. 잡코리아 기업 회원 로그인
2. 채용공고 등록 (미등록 시 공고 등록 먼저 진행)
3. **공고 관리** 메뉴에서 해당 공고의 **[유료서비스 신청]** 클릭
4. 원하는 상품 선택 및 이용 기간 설정
5. 결제 완료 후 즉시 노출 시작

### 8.3 프리미어 채용관 구매

- 프리미어 채용관은 **전담 컨설턴트를 통해서만** 구매 가능합니다
- 고객센터로 문의하여 전담 컨설턴트 배정을 요청하세요
